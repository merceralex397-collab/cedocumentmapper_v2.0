# EPIC-01 Domain Contracts

## Objective

Finalize Python models and JSON schemas that all modules share.

## Deliverables

- Document model dataclasses or Pydantic models.
- Field order and required-field constants.
- Contract tests for JSON schemas.
- Typed issue/result models.

## Acceptance Criteria

- EVA export schema validates the known final JSON example.
- Expected fixture schema validates sample expected files.
- Contract docs and Python models agree on field names.
