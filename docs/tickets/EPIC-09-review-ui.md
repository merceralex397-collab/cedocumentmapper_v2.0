# EPIC-09 Review UI

## Objective

Build the user-facing review and provider setup workflow.

## Deliverables

- Drag-and-drop import.
- Field review panel.
- Source preview with source-span highlighting.
- Provider diagnostics panel.
- Provider setup/editor.
- Batch mode.
- Engineer report overlay support.

## Acceptance Criteria

- UI can load a record produced by the application service.
- Editing fields updates export values without re-running extraction.
- Engineer report overrides are visibly highlighted.
- Parser modules remain UI-independent.
