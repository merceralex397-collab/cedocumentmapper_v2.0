# EPIC-11 Current Corpus Import

## Objective

Turn existing sample documents into a v2 regression corpus.

## Deliverables

- Inventory of current samples by provider and file type.
- Expected fixture files for each usable sample.
- Redaction notes where needed.
- Initial pass/fail report against v2 readers and rules.

## Acceptance Criteria

- Every active provider has at least one fixture.
- Engineer report fixtures cover overlay behavior.
- Image-only documents are classified separately from instruction documents.
- Fixture gaps are tracked as follow-up tickets.
